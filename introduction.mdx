The BloxGen Public API allows you to programmatically generate Roblox accounts using your API key. This API provides the same functionality as the web dashboard but designed for automation and integration.

## Base URL

```
https://api.bloxgen.net
```

## Account Types

| Type | Description |
|------|-------------|
| `alt` | Fresh alternative accounts |
| `+30 days old` | Accounts aged 30+ days |
| `+1 year old` | Accounts aged 1+ year |
| `5+ years old` | Accounts aged 5+ years |
| `dump` | Verified dump accounts |
| `dump (unchecked)` | Unchecked dump accounts |

## Quick Example

```bash
curl -X POST https://api.bloxgen.net/api/generate \
  -H "Content-Type: application/json" \
  -d '{"apiKey": "BLOX-XXXXXXXXXXXXXXXX", "type": "alt"}'
```

```json
{
  "success": true,
  "data": {
    "username": "example_user",

  }

```
